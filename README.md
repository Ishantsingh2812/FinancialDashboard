📊 Financial Dashboard

A modern, full-stack financial dashboard built with Next.js App Router, designed to provide an intuitive interface for tracking invoices, revenue, and customer data in real time.

This project is built upon the official Next.js App Router Course, extended with custom features and enhancements.


🚀 Overview

Financial Dashboard is a web application that allows users to:


View revenue trends and financial summaries at a glance
Manage and track customer invoices
Search, filter, and paginate through financial records
Experience fast, server-rendered performance powered by Next.js
